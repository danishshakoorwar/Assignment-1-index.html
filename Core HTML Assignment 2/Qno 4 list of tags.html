List of HTML Tags

  1. <html>: The root element that wraps the entire HTML document.
  2. <head>: Contains meta-information about the HTML document, such as title, character set, etc.
  3. <title>: Sets the title of the HTML document, displayed in the browser's title bar or tab.
  4. <body>: Contains the content of the HTML document, such as text, images, links, etc.
  5. <h1> to <h6>: Headings of different levels, where <h1> is the highest and <h6> is the lowest.
  6. <p>: Represents a paragraph of text.
  7. <a>: Defines a hyperlink, allowing you to link to other documents or resources.
  8. <img>: Embeds an image in the HTML document.
  9. <ul>: Represents an unordered list, typically used with <li> elements.
  10. <ol>: Represents an ordered list, typically used with <li> elements.
  11. <li>: Represents a list item within <ul> or <ol>.
  12. <table>: Defines a table, typically used with <tr>, <td>, and <th> for rows, cells, and headers.
  13. <form>: Creates an HTML form that can be used to collect user input.
  14. <input>: Represents an input field, such as text input, checkboxes, radio buttons, etc.
  15. <div>: Defines a division or a section in an HTML document, often used for styling with CSS.
  16. <span>: Inline equivalent of the <div> element, used for styling smaller sections of text.
