1. HTML

    Primary Use Cases Key Features/Benefits
    The primary use case of HTML (HyperText Markup Language) is to structure and define the content of a web page. HTML provides a set of elements and tags that allow you to organize information on a webpage, making it understandable for web browsers
    • Simple and easy to learn
    • Compatible with all web browsers
    • Allows for semantic markup

  2. CSS

    Primary Use Cases Key Features/Benefits
    The primary use case of CSS (Cascading Style Sheets) is to style and format the visual presentation of HTML documents on the web. CSS allows web developers to control the layout, colors, fonts, spacing, and other visual aspects of a webpage.
    • Allows for separation of content and presentation
    • Enables responsive design
    • Offers a wide range of styling options

  3. JavaScript

    Primary Use Cases Key Features/Benefits
    The primary use case of JavaScript is to add interactivity and dynamic behavior to web pages. JavaScript is a versatile programming language that runs on the client side (in the user's web browser) and allows developers to create dynamic, interactive, and responsive web applications.
    • Can manipulate and modify web page content in real-time
    • Offers a wide range of functionality through libraries and frameworks
    • Allows for server-side scripting with Node.js
